iTracker

  • Project Abstract: A GPS based Tracking system for Institutional vehicles and Students
  • Project date: May 2024
  • Project website: Cells Monitoring website
  • Project Application: Equinox
  • Technologies used:
    1. GPS Module - Adafruit Ultimate GPS Breakout Board
    2. internet of Things - MQTT
    3. Microcontroller programming Embedded C
    4. Radio-Frequency Identification (RFID)
    5. Data Analysis

Project Overview

  • Significance:
    • Develop a comprehensive tracking system for Kumaraguru College buses and students to enhance safety and management efficiency.
  • System Overview:
    • Bus Tracking: Each college bus is equipped with a GPS tracker.
    • Student Tracking: Students are tracked using RFID technology integrated with their identity cards.
  • Data Transmission:
    • MQTT Technology: Data from GPS trackers and RFID readers is transmitted using MQTT (Message Queuing Telemetry Transport) technology.
    • HiveMQ Servers: The transmitted data is sent to HiveMQ servers for processing and analysis.
  • Features and Benefits:
    • Real-time Tracking: Provides real-time location updates of college buses.
    • Attendance Management: Automates student attendance tracking through RFID, reducing manual errors.
    • Safety: Enhances student safety by monitoring their presence in buses and at designated stops.
    • Efficiency: Improves the efficiency of the college transportation system by optimizing routes and schedules.
  • Technical Implementation:
    • GPS Trackers: Installed in buses to continuously send location data.
    • RFID Readers: Placed at bus entry points to scan student identity cards.
    • MQTT Protocol: Ensures efficient, low-latency data transmission to HiveMQ servers.
    • HiveMQ: A robust MQTT broker that handles incoming data, facilitating real-time tracking and analytics.
  • Outcomes:
    • Enhanced Safety: Increased security for students through continuous monitoring.
    • Operational Efficiency: Streamlined transportation operations and improved resource management.
    • Data-Driven Insights: Provides valuable insights into bus utilization and student attendance patterns.
